# SHAME — Bar · Yerevan

Одностраничный лендинг бара **SHAME** (Ереван). Тёмная редакторская эстетика, крупная типографика, бегущие строки, скролл-анимации, мультиязычность (RU / EN / HY) и реальная карта Google.

🔗 **Прод:** https://rybushkin.github.io/shame-bar/
📦 **Репозиторий:** https://github.com/rybushkin/shame-bar

---

## Что это

- **Единственный файл — [`index.html`](index.html)**. Никакой сборки, фреймворков и зависимостей. Чистый HTML + CSS (инлайн-стили) + один `<script>` на ванильном JS.
- Изначально импортировано из Claude Design (`SHAME.dc.html`, формат Design Component) и переведено в самостоятельную страницу без проприетарного рантайма `support.js`.
- Шрифты: **Syne** (заголовки), **Space Grotesk** (текст), **Space Mono** (подписи/моно) — с Google Fonts.
- Палитра: фон `#0F0F10` / вторичный `#171717`, текст `#F5F5F2`, акцент `#E4513C` (красный), приглушённый `#8E8E8E`.

## Запуск локально

Статический файл — достаточно любого сервера:

```bash
cd "путь/к/shame-bar"
python3 -m http.server 8137
# открыть http://localhost:8137/
```

В Claude Code есть готовый конфиг превью — [`.claude/launch.json`](.claude/launch.json) (имя сервера `shame-static`, порт 8137).

> ⚠️ Анимации (бегущие строки) и сторонние iframe (карта Google) **не отображаются во встроенном превью Claude** — вкладка фоновая, браузер ставит CSS-анимации на паузу, а кросс-домен iframe не рендерится. Проверять всегда в обычном браузере.

## Деплой (GitHub Pages)

Хостинг — GitHub Pages, ветка `main`, корень. Публикуется **только `index.html`** (исходники дизайна и локальные конфиги исключены через [`.gitignore`](.gitignore)).

```bash
git add index.html && git commit -m "..." && git push
# Pages пересоберётся автоматически (~1 мин)
```

Проверить статус сборки и живую версию:

```bash
gh api repos/rybushkin/shame-bar/pages/builds/latest --jq .status   # ждём "built"
curl -s "https://rybushkin.github.io/shame-bar/?v=$(date +%s)" -H 'Cache-Control: no-cache' | head
```

Pages был включён однократно так:
```bash
gh api -X POST repos/rybushkin/shame-bar/pages -f "source[branch]=main" -f "source[path]=/"
```

## Мультиязычность (i18n)

- Языки: **RU (по умолчанию) / EN / HY (армянский)**.
- Весь переводимый текст — в словаре `I18N` внутри `<script>` в `index.html` (объекты `ru` / `en` / `hy`).
- В разметке у переводимых элементов стоит атрибут `data-i18n="ключ"`; функция `applyLang()` подменяет `innerHTML` по словарю, меняет `<title>`, `meta description` и `<html lang>`.
- Переключатель — кнопки `data-lang-btn` (в шапке первого экрана и в плавающем меню). Выбор сохраняется в `localStorage` (`shame-lang`), при первом заходе определяется по языку браузера.
- **Текст по умолчанию в HTML — русский** (для случая без JS).

### Как менять текст
1. Поправить значение в HTML (русский дефолт) **и** соответствующий ключ во всех трёх языках в словаре `I18N`.
2. Брендовые элементы (`SHAME`, `SHAME BAR`, `@shame.bar`, координаты, цены в драмах) переводить не нужно.
3. `id`/якоря секций (`#drinks`, `#evidence`, `#visit` …) — оставлять латиницей, на них завязаны навигация и плавный скролл.

## Карта Google (тёмная)

В секции «Контакты» — встроенная карта через **share-embed** (`https://www.google.com/maps/embed?pb=…`) с пином на место Shame (place_id `0x406abcfedd5b85e5:0x819dd30116528d72`, координаты `40.1788505, 44.5062254`).

- Тёмная тема — CSS-фильтр `.map-dark { filter: invert(92%) hue-rotate(180deg) … }` поверх iframe (без API-ключа).
- ⚠️ **Старый трюк `…/maps?q=…&output=embed` больше не работает** (отдаёт 404 + `X-Frame-Options: SAMEORIGIN`). Используем только `/maps/embed?pb=…`.
- Чтобы получить корректный `pb` для нового адреса: Google Maps → место → **Поделиться → Встроить карту** → скопировать `src` из `<iframe>`. Либо подобрать `pb` так, чтобы `curl` к URL отдавал HTTP 200 (валидный pb → 200; кривой → 404).
