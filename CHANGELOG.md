# Changelog

Все заметные изменения проекта SHAME Bar. Даты в формате ГГГГ-ММ-ДД.

## 2026-06-29

### Интро-лоадер
- Добавлен стартовый экран загрузки: слово **SHAME** контуром, по которому слева направо «заливается» красным акцентом (`#E4513C`), затем оверлей плавно растворяется и открывает сайт.
- Реализация: CSS-анимации `loader-fill` (clip-path wipe) + `loader-hide` (fade/visibility); JS-страховка гарантированно убирает лоадер (на случай отключённых/паузнутых анимаций); учитывается `prefers-reduced-motion`.

### Проектная документация
- Добавлены `README.md`, `CHANGELOG.md`, `TODO.md`.

## 2026-06-29 (ранее)

### Instagram + карта (`6e39602`)
- Добавлена секция **Instagram** (`#instagram`, № 08) — заголовок, кнопка `@shame_bar`, сетка из 6 плиток-ссылок на профиль (плейсхолдер ленты).
- Заглушка карты в «Контактах» заменена на **реальную тёмную карту Google** через share-embed (`/maps/embed?pb=…`) с пином на место Shame; тёмная тема через CSS-фильтр; кнопка «Открыть в Google Maps».
- Контакты перенумерованы 08 → 09; чередование фонов секций сохранено.
- Зафиксировано: старый `…/maps?q=…&output=embed` больше не фреймится (404 + SAMEORIGIN).

### Переключатель языков RU/EN/HY (`04fa0dc`)
- Добавлена система i18n: словарь `I18N` (ru/en/hy), атрибуты `data-i18n`, функция `applyLang()`.
- Переключатель языков в шапке первого экрана и в плавающем меню; синхронные активные состояния.
- Полный перевод на **армянский**; добавлен EN обратно как один из языков (английский не удалялся).
- Выбор языка сохраняется в `localStorage`, автоопределение по языку браузера; обновляются `<title>`, `meta description`, `<html lang>`.

### Перевод на русский (`7f6ced1`)
- Весь видимый текст переведён на русский (дефолтный язык страницы).

### Первый релиз (`7c224d0`)
- Импорт дизайна из Claude Design (`SHAME.dc.html`) и реализация в самостоятельный `index.html` без рантайма `support.js`.
- Перенос `<helmet>` в `<head>`; компиляция `style-hover` в реальные CSS `:hover`-правила; порт логики `componentDidMount` (reveal по скроллу, параллакс, фейд навигации, плавный скролл).
- Бегущая строка-лого «SHAME BAR ✱» на первом экране (бесконечная, бесшовная) и аутлайн-версия в подвале.
- Добавлена секция меню с авторскими коктейлями (тестовый контент).
- Развёрнуто на GitHub Pages (ветка `main`, публикуется только `index.html`).
